# 🎨 Заміна логотипу StakeYourTake

## 📁 Що потрібно зробити:

### 1. **Додай свій логотип**
Помісти свій файл логотипа в папку:
```
public/logo.png
```

### 2. **Рекомендовані параметри логотипа:**

#### 📏 **Розмір:**
- **Мінімум:** 40x40px
- **Оптимально:** 80x80px або 120x120px
- **Формат:** квадратний або трохи прямокутний

#### 🎨 **Формат та якість:**
- **Найкраще:** SVG (масштабується без втрати якості)
- **Добре:** PNG з прозорістю
- **Якість:** висока, чіткі краї

#### 🌈 **Колір:**
- **Рекомендовано:** білий або золотий логотип
- **Контраст:** добре видно на будь-якому фоні
- **Прозорість:** якщо є прозорі ділянки

## 🔧 **Як це працює:**

### ✅ **Автоматична заміна:**
- Коли додаш `public/logo.png` - він автоматично замінить іконку Sparkles
- Якщо логотип не завантажиться - показується запасна іконка

### 🎯 **Позиціонування:**
- **Розмір:** `w-8 h-8` (32x32px)
- **Позиція:** ліворуч від тексту "StakeYourTake"
- **Анімація:** пульсація збережена

### 🔄 **Fallback система:**
- Якщо `logo.png` не знайдено - показується Sparkles
- Автоматичне переключення при помилці завантаження

## 🎨 **Дизайн поради:**

### ✅ **Що працює добре:**
- Простий, чистий логотип
- Високий контраст
- Чіткі краї
- Прозорий фон

### ❌ **Чого уникати:**
- Складні деталі (можуть бути нечитабельними)
- Темний логотип на темному фоні
- Великі розміри файлів
- Низька роздільна здатність

## 🚀 **Готово!**

Після додавання файлу `public/logo.png`:
1. Перезавантаж сторінку
2. Твій логотип з'явиться замість іконки Sparkles
3. Збережеться пульсація логотипа
4. Автоматичний fallback якщо щось піде не так

## 📱 **Мобільна версія:**
Логотип автоматично адаптується для мобільних пристроїв та зберігає пропорції.

**Тепер твій сайт матиме унікальний логотип!** 🎨✨
