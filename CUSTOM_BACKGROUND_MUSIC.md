# 🎨 Кастомний фон та фоновий музичний супровід

## 📁 Що потрібно зробити:

### 1. Додай свій PNG фон
Помісти свій файл у:
```
public/custom-background.png
```

### 2. Додай фонову музику
Помісти музичний файл у:
```
public/background-music.mp3
```

### 3. Оптимізація файлів

#### 🖼️ Для фону:
- **Розмір**: 1920x1080px або більше
- **Формат**: PNG з прозорістю
- **Вага**: до 2MB
- **Колірна схема**: темна для кращої читабельності

#### 🎵 Для музики:
- **Формат**: MP3
- **Тривалість**: 2-3 хвилини (буде повторюватись)
- **Якість**: 128kbps достатньо
- **Розмір**: до 1MB

## 🎵 Як працює музика:

### Автоматичне відтворення:
- **Спроба стартувати** при завантаженні сторінки
- **Блокується браузером** → чекає на взаємодію користувача
- **Стартує** після першого кліку/натискання клавіші

### Контролери:
- **Кнопка музики** в лівому нижньому куті
- **Play/Pause** керування
- **Гучність** 0-100%
- **Статус** відображення

## 🎨 Налаштування фону:

### CSS властивості:
```css
.bg-bg-primary {
  background-image: url('/custom-background.png');
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
}
```

### Ефекти:
- **Parallax** для фіксованого фону
- **Overlay** для кращої читабельності
- **Responsive** адаптація для мобільних

## 🔧 Налаштування:

### Змінити музику:
```typescript
// У файлі src/services/backgroundMusic.ts
this.audio.src = '/your-music-file.mp3';
```

### Змінити гучність за замовчуванням:
```typescript
// У файлі src/services/backgroundMusic.ts
this.volume = 0.5; // 50% замість 30%
```

### Вимкнути автоплей:
```typescript
// Закоментуй рядок у initAudio()
// this.play();
```

## 📱 Мобільна оптимізація:

- **Фон** стає scroll-friendly на мобільних
- **Музика** вимикається при економії батареї
- **Контролери** адаптивні для touch-пристроїв
