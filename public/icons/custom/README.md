# Custom Icons

Add your custom avatar icons here (64x64px recommended)

## Supported formats:
- PNG (with transparency)
- SVG (vector, scalable)
- WebP (modern format)

## Naming convention:
- avatar-01.png
- avatar-02.svg  
- bitcoin-bull.png
- eth-crystal.svg

## Usage:
Icons will be automatically loaded from /icons/custom/ directory
