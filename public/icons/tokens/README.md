# Token Icons

Add your custom token icons here (64x64px recommended)

## Supported formats:
- PNG (with transparency)
- SVG (vector, scalable)
- WebP (modern format)

## Naming convention:
- btc.png - Bitcoin icon
- moto.png - MOTO token icon
- piil.png - PIIL token icon

## Usage:
Icons will be automatically loaded from /icons/tokens/ directory
